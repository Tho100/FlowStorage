### Flow Storage - Awesome file hosting service
#### Flow Storage is awesome bcause it allows you to upload & manage your file from any devices with supports of creating multiple accounts
#

### Requirements
- 25MB Disk Space
- Internet Connection 

### Operating System
- Windows 10 
- Mac OS
- Android 
- iOS
